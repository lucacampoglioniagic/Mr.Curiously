Mr. Curiously Privacy Policy

Last updated: May 2026

Mr. Curiously ("the App", "we", "us") is an automated social media content publishing application. This Privacy Policy explains how Mr. Curiously handles data in connection with its services, including its integration with TikTok and Instagram.

1. Data Collection

Mr. Curiously does not collect, store, or process any personal data from end users or third parties. The App operates as a single-account automation tool owned and operated solely by its creator. No user registration, sign-up, or personal information submission is required or possible.

2. TikTok and Instagram Integration

Mr. Curiously uses OAuth 2.0 tokens to publish content on behalf of a single, owner-controlled TikTok and Instagram account. The authorization flow (e.g., "Connect with TikTok") is available on our website and is used exclusively by the App owner to grant Mr. Curiously permission to post content. No third-party users are asked to authenticate or share their credentials. OAuth tokens are stored securely as encrypted environment variables and are never shared with any third party.

3. Content Posting API

Mr. Curiously uses the TikTok Content Posting API to automatically publish short-form video content to a single TikTok creator account. The App accesses only the permissions required for content publishing (video upload and post creation). No data is read from TikTok users' accounts beyond what is strictly necessary for authentication confirmation.

4. Cookies and Tracking

This website does not use cookies, tracking pixels, or any form of user analytics.

5. Third-Party Services

Mr. Curiously interacts with the following third-party APIs:

  • TikTok – Content Posting API for video publishing
  • Instagram (Meta) – Graph API for image and reel publishing
  • OpenAI / Anthropic – AI language models for caption generation
  • Cloudflare R2 – Cloud storage for media assets

Each third-party service is governed by its own privacy policy and terms of service.

6. Data Security

All API credentials and access tokens used by Mr. Curiously are stored as encrypted environment secrets (e.g., GitHub Actions Secrets) and are never exposed in source code or logs.

7. Changes to This Policy

We may update this Privacy Policy from time to time. All changes will be reflected on this page with an updated date.

8. Contact

For any questions regarding this Privacy Policy or Mr. Curiously, contact us via the GitHub repository: github.com/lucacampoglioniagic/Mr.Curiously
